# Project-Pandora

> Open de box, Face the Bugs.
 
Pandora is an experimental procedural debugging and fault injection platform designed to help developers improve debugging skills through realistic software failures.

Instead of simply breaking code, Pandora injects plausible and procedural bugs that simulate real-world development problems — from logic flaws and silent failures to concurrency issues and unpredictable behavior.

---

## Vision

Debugging is one of the most valuable skills in software engineering, yet most developers only learn it through accidental failures.

Pandora aims to transform debugging into a deliberate practice

The project focuses on:
- Procedural bug injection
- Debugging training
- Chaos engineering concepts
- Software mutation
